# 🎨 KIDS PHOTO PRINTING STUDIO

# 

# The simplest way for kids to create, arrange, and print photo collages.

# 

# KIDS PHOTO PRINTING STUDIO is a lightweight, open-source Windows application designed specifically for children. Unlike Word or Photoshop, which can be frustrating with hidden layers and text-wrapping rules, this app provides a pure "blank paper" canvas.

# 

# Kids can drag, drop, and arrange photos with zero technical friction.

# 

# 🚀 Key Features

# 

# 🛠️ Kid-Proof Interface

# 

# Massive Buttons: Large, color-coded buttons for Paste, Print, and Undo.

# 

# Contextual Tools: Delete, Rotate, and Crop buttons only appear when a photo is selected.

# 

# Visual Feedback: Large blue handles and mouse-wheel support for resizing.

# 

# 🖼️ Advanced Image Handling

# 

# Drag \& Drop: Pull photos directly from Windows folders onto the paper.

# 

# Wide Format Support: Works with .jpg, .png, .gif, .webp, .bmp, .tiff, and even iPhone .heic/.heif files.

# 

# 4K Smart-Shrink: Massive photos are automatically resized on import to stay fast and centered.

# 

# ✂️ Creative Freedom

# 

# Visual Crop: Click "Crop" and draw a red box directly on the photo to trim it.

# 

# One-Click Rotate: Spin images 90° instantly.

# 

# Unlimited Undo: Total safety with the Undo button or Ctrl + Z.

# 

# A4 Optimized: Perfectly maps to standard A4 paper for easy printing.

# 

# 💻 Requirements

# 

# Operating System: Windows 10 or 11.

# 

# Language: Python 3.x.

# 

# Libraries: PyQt6.

# 

# 🛠️ Installation

# 

# Clone this repository:

# 

# git clone \[https://github.com/YOUR\_USERNAME/kids-photo-studio.git](https://github.com/YOUR\_USERNAME/kids-photo-studio.git)

# 

# 

# Install the required library:

# 

# pip install PyQt6

# 

# 

# Run the app:

# 

# python kids\_studio.py
